# VALCOACH V3

Application locale de coaching Valorant.

## Installation locale

1. Télécharge le projet.

2. Installe les dépendances :

```bash
npm install
```
3. Complète le fichier .env avec tes informations :
```
RIOT_API_KEY=ta_cle_riot_api_ici
TRACKER_API_KEY=ta_cle_tracker_api_ici
VALORANT_REGION=eu
VALORANT_NAME=TonPseudo
VALORANT_TAG=TonTag
```
4. Lance l'application avec le fichier :
```
restart-valcoach.bat
```
5. Une page internet s’ouvrira automatiquement à cette adresse :
```
http://localhost:3000

```
ATTENTION : Le fichier `.env` ne doit jamais être envoyé sur GitHub. Il contient tes clés API privées!!!!!!!!!!!!
