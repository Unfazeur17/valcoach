VALCOACH Project Overview :

*VALCOACH is a public-facing Valorant training dashboard concept that helps players review their own match performance and build training plan

Purpose :

*This is a project for personal improvement in Valorant. The goal is to turn match results and recurring mistakes into concrete training objectives instead of only tracking RR

*The dashboard is not a commercial product, does not sell data, and is not automated gameplay
Rank and RR summary for personal progression tracking :

*Manual match logging for result, agent, map, RR, ACS, K/D/A, first kills, first deaths, headshot percentage, and avoidable deaths

*Personal self-review scores for crosshair placement, peek discipline, communication, tilt management, and utility usage

*Death reason tagging, including timing mistakes, bad peeks, poor angles, utility impact, stun, mechanical errors, untraded deaths, economy mistakes, and tilt

*VOD review notes attached to each game

*Training log with date, training hours, exercise type, and specific exercises

*Agent/map dashboard and coaching goals generated from ia patterne

*Macro library by agent with image or video references, attack/defense labels, title, and description
Privacy and Security

*API keys are stored in a local .env file and are not exposed in the browser code

*Imported data is used for personal performanc analysis and training planning.

*After a successful first version, I would like to publish a public version of VALCOACH.
